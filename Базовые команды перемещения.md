Базовые команды для пермещения в файловой системе linux

Вот базовые команды Linux для перемещения по файловой системе (и пару полезных “рядом стоящих”):

### Где я сейчас

* `pwd` — показать текущий каталог (полный путь)

### Посмотреть содержимое

* `ls` — список файлов/папок
* `ls -l` — подробный список
* `ls -a` — включая скрытые (начинаются с `.`)
* `ls -lah` — удобно: подробно + скрытые + “человеческие” размеры

### Перейти в каталог

* `cd ПАПКА` — перейти в папку
* `cd ..` — на уровень выше
* `cd` или `cd ~` — в домашний каталог
* `cd /` — в корень
* `cd -` — вернуться в предыдущий каталог

### Быстрые подсказки по путям

* `.` — текущий каталог
* `..` — родительский каталог
* `~` — домашняя папка пользователя
* Абсолютный путь: начинается с `/` (например, `/etc`)
* Относительный путь: без начального `/` (например, `Documents`)

### Создать/удалить каталоги (часто нужно при навигации)

* `mkdir имя_папки` — создать папку
* `mkdir -p a/b/c` — создаст полный путь папок a/b/c
* `rmdir имя_папки` — удалить пустую папку
* `rm -r имя_папки` — удалить папку с содержимым (**осторожно**)
* `rm -rf имя_папки` — принудительно, без вопросов (**очень осторожно**)

### Полезные для “ориентации”

* `tree` — показать дерево каталогов (если установлено)
* `find . -name "имя"` — найти файл/папку по имени начиная с текущей
* `locate имя` — быстрый поиск (если настроена база)
* `echo $HOME` — путь к домашнему каталогу
* `which команда` — где находится исполняемый файл команды

### Мини-шпаргалка примеров

```bash
pwd
ls -lah
cd /etc
cd ..
cd ~
cd -
```
