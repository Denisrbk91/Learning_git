Chown - команда для смены владельца 

chown НОВЫЙ_ВЛАДЕЛЕЦ файл_или_папка
Пример : sudo chown alice report.txt

Смена владельца и группы
sudo chown alice:devs mydir
владелец станет `alice`
группа станет devs 
 Если указать только `:группа`, владелец не меняется, меняется только группа:
 
 `-R` Применяет ко всем файлам/подпапкам внутри:
 
 sudo chown -R alice:devs /srv/project
