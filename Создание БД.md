`systemctl start postgresql` — запустить сервис PostgreSQL  
`systemctl status postgresql` — проверить статус сервиса  
`systemctl enable postgresql` — добавить PostgreSQL в автозапуск

`sudo -u postgres psql` — войти в PostgreSQL под пользователем `postgres`

`CREATE DATABASE netbox;` — создать базу данных  
`CREATE USER netbox WITH PASSWORD '111';` — создать пользователя БД  
`GRANT ALL PRIVILEGES ON DATABASE netbox TO netbox;` — выдать пользователю права на БД

`\q` — выйти из psql

`psql -U netbox -h localhost -W netbox` — подключиться к БД под пользователем `netbox`

`\l` — список всех баз данных  
`\du` — список пользователей PostgreSQL
