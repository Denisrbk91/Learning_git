ip a - посмотреть информацию об айпи
ip addr flush dev ensXXXX  - Очистить сетевой интерфейс от всех адресов
ip addr add 192.168.X.X/(маска) dev ensXXX - назначить адрес(до ребута)

НАСТРОЙКА СЕТИ
/etc/network/interfaces - здесь происходит настройка сети ,открывается через vim 
iface ensXX inet dhcp - здесь меняем dhcp на static 
address xx.xx.xx.xx
netmask xx.xx.xx.xx

Поднять сеть : ip link set ensXX up

Пример,не забываем править ens 
auto ens160
iface ens160 inet static
    address 192.168.0.1
    netmask 255.255.255.0
